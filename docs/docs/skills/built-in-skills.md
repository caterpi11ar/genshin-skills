# 内置技能详情

## welkin-moon — 月卡每日领取

- **ID**: `welkin-moon`
- **说明**: 登录 → 启动游戏 → 领取月卡奖励
- **执行顺序**: 第一个执行，负责启动云原神游戏

此技能打开云原神页面，完成登录流程，启动游戏后领取月卡每日奖励。后续技能从游戏内界面继续操作。

## claim-mail — 邮件领取

- **ID**: `claim-mail`
- **说明**: 打开邮箱 → 一键领取所有附件

在游戏内打开邮箱界面，检查是否有未读邮件，如有则一键领取所有附件。

## expedition-collect — 探索派遣收取

- **ID**: `expedition-collect`
- **说明**: 冒险之证 → 收取已完成派遣 → 重新派遣

打开冒险之证界面，收取所有已完成的探索派遣奖励，然后重新派遣角色。

## battle-pass-claim — 纪行奖励领取

- **ID**: `battle-pass-claim`
- **说明**: 打开纪行 → 领取可领取的等级奖励

打开纪行界面，领取所有可领取的等级奖励。
